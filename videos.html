Amin page

This is some of the stuff that I do:

  1. create entries for the dojos on meetup.com
  2. agree about an agenda/kata for the dojo and update the agenda in meetup
  3. announce the dojo on meetup
  4. before a dojo prepare a repo for it:
    1. create a repo for the dojo code on https://github.com/scala-vienna say I call it https://github.com/scala-vienna/buzzfizz
    2. git clone git@github.com:scala-vienna/buzzfizz.git
    3. cd buzzfizz
    4. git remote add quickstart git@github.com:scala-vienna/dojo-quickstart.git
    5. git pull quickstart master
    6. git push -u origin master
    7. git remote rm quickstart
  5. bring a laptop with mouse and VGA adapter to the dojo
  6. during the dojo: make sure people have a good time
  7. after the dojo: push the code to GitHub
